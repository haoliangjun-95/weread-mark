# 关于微痕

微痕是一款微信读书数据可视化工具，帮你更直观地了解自己的阅读习惯。只需在浏览器中打开，输入你的微信读书 API Key，就能一站式查看阅读数据、管理书架笔记。

> 感谢微信读书团队让有趣的灵魂在茫茫人海中相遇！

---

## 快速开始

1. 在浏览器中打开微痕
2. 进入「设置」页面，输入你的微信读书 API Key，点击保存
3. 保存成功后自动跳转到阅读看板，开始探索你的阅读数据

> 获取 API Key：前往微信读书网页版获取。

[含图文的功能介绍](https://blog.haoliangjun.com/archives/wei-hen-gong-neng-jie-shao-NTUROhxO)

## 📊 阅读看板

一站式查看你的阅读数据概览。

- **切换时间范围**：本周、本月、某一年、总计，灵活选择想看的时间段
- **核心统计**：读完了几本书、总共花了多少时间、平均每天读多久，一目了然
- **阅读趋势**：最近 30 天阅读时长折线图、各月阅读量柱状图，直观看到阅读节奏的变化
- **时段分布**：一天中哪个时段你最常读书？早鸟还是夜猫子，数据告诉你
- **偏好分析**：你最常读哪类书？偏爱哪些作者？阅读时长最长的书是哪本？

---

## 🔍 全文搜索

在你的所有划线和想法中按关键词搜索，快速找到某句话、某个观点出自哪本书。

- 输入关键词，系统实时扫描所有笔记
- 搜索过程中可随时中止
- 点击搜索结果直接跳转到对应笔记，并自动高亮定位

---

## 📚 书架

管理你的微信读书书架，分为「我的书架」和「已读完书架」两个视图。

- 卡片式展示，按书名或作者搜索、按分类筛选
- 每本书显示划线数、想法数、书评数
- 点击数字标签可直接跳转查看对应的笔记内容
- 滚动自动加载更多，不用翻页

---

## 📝 笔记

集中管理所有读书笔记，支持列表浏览和单书详情。

### 列表浏览
- 搜索、筛选、滚动加载，快速定位目标书籍
- 每张卡片底部有「📘 成书」按钮，一键将这本书的所有划线、想法和书评汇总导出为排版精美的 PDF

### 书籍详情
- **划线与想法**：查看每条划线，以及你在划线旁写下的想法，关联展示
- **书评**：查看你给这本书写的书评，支持逐条导出为图片或 PDF

### 笔记成书
想把一本书的精华整理成文档？点击「📘 成书」：

- 自动汇总该书的全部划线、想法和书评
- 生成精美的排版页面，包含书籍封面
- 通过浏览器打印即可保存为 PDF，默认文件名 `书名-笔记成书.pdf`

---

## ⭐ 书评墙

浏览你写过的所有书评，按时间倒序排列。

- 自动扫描所有书评，后续访问有缓存，加载更快
- 支持按书名/作者搜索，滚动加载更多
- 点击「预览」弹窗全屏阅读，享受沉浸式阅读体验
- 每条书评可单独导出为图片或 PDF，默认文件名 `书名-书评.pdf`

---

## ⚙️ 个性化设置

微痕提供丰富的个性化选项，让界面更符合你的审美和阅读习惯。

### 主题
8 种配色方案随心切换：默认、深色、护眼、浅绿、纸白、冷灰、杏暖、月粉。夜间阅读推荐深色或护眼模式。

### 字体
8 种中文字体可选：霞鹜文楷、思源宋体、无衬线、方正悠宋、京华老宋体、思源黑体、马山正楷、站酷小薇体。

### 字号
自动适配屏幕大小，也可手动选择小、中、大、超大。

### 导出风格
笔记和书评导出图片时，可选择经典白、暖纸、深色三种风格。

### 阅读箴言
随机展示名人名言，为阅读增添灵感。

---

## 🔒 隐私说明

- 你的 API Key 仅保存在浏览器本地（localStorage），不会上传到任何服务器
- 所有数据直接从微信读书官方 API 获取，微痕不存储你的阅读数据
- 清除浏览器数据后需要重新输入 API Key
